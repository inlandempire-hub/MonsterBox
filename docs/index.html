A note on PDF importing

Text-based import is the main system, and it's working well across a wide range of books, but it's still being actively tested. Most stat blocks come through cleanly; a few are flagged for a quick review so you can confirm them before use.

OCR import is experimental. When a PDF's stat blocks are images, or use a font the text layer can't read, MonsterBox offers an OCR pass as a last resort. It reads the page like a scanner, good enough to turn an otherwise unreadable PDF into something you can review and fix, but not as accurate as text import. Expect typos and to-be-checked fields; every OCR'd creature is sent to the review queue with a screenshot of its page beside the editor.

In short: if a book imports cleanly, great. If it doesn't, OCR is there to rescue what it can. Either way, give flagged entries a quick look before play.
